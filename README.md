# DevSecOps Pipeline CI com Python & Bandit 

#### Este projeto é uma demonstração prática de implementação de um pipeline DevSecOps completo para o app Python, utilizando GitHub Actions como ferramenta de integração contínua (CI). <br />
<img width="944" height="699" alt="image" src="https://github.com/user-attachments/assets/4322f916-0d3c-4c9c-8cdb-083fbbbbccf1" />




#### O objetivo principal é ilustrar como automatizar verificações de segurança no ciclo de vida de desenvolvimento de software, promovendo boas práticas de DevSecOps.

## O pipeline realiza:

#### - Análise estática de código (SAST) com Bandit

#### - Automação de processos de segurança para integração contínua (CI).

#### - Execução em contêiner/runner seguro.


## Habilidades técnicas (Hands-on)
- DevSecOps / CI

- Configuração de GitHub Actions para integração contínua.

- Criação de pipelines automáticos de segurança.

- Entendimento de workflows CI/CD, triggers (push, branches) e jobs.

## Segurança de código

- Uso do Bandit para SAST.

- Identificação de vulnerabilidades em código Python (ex.: host='0.0.0.0').

- Aprendizado sobre boas práticas de segurança em aplicações web.

- Python / Desenvolvimento
  
- Estruturação de projeto Python com requirements.txt.

- Compreensão do ambiente virtual e dependências.

- Gerenciamento de dependências
